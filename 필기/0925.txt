**객체와 Class 정리

Class란 개발자가 표현하려는 대상
멤버변수와 메서드로 구성
멤버변수는 값을 저장
메서드는 기능구현

Class와 객체의 관계
-Class를 사용하기 위해 객체로 만든다.
-사용: 멤버변수(값을 저장), 메서드(기능구현)를 사용
-생성: new 연산자를 통해>> new 생성자
-생성자: 객체가 생성될 때 제일 먼저 실행되는 메서드
	호출이 불가능
	생성자의 이름은 class와 동일
객체는 heap영역에 저장
참조변수는 저장된 객체의 주소를 저장


**ArrayList
-배열: 고정길이, 순차적 자료저장(인덱스 순서가 있음) 
	>선언: String [ ] a=new String [3];

-List: "가변길이", 순차적 자료저장(인덱스 순서가 있음)
	>선언: ArrayList <"저장할 타입">b=new ArrayList<>();
